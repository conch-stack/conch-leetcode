##### Big O notation

- O(1) 常数复杂度
- O(logn) 对数复杂度： 二分查找法
  - 平衡二叉树
- O(n) 线性复杂度 ：二叉树遍历（每个节点只遍历一次）   ，排序(Optimal sorted matrix search)
- o(nlogn) 线性对数复杂度：快排，归并排序
-  O(n^2) 平方 ： 斐波那契
- O(n^3) 立方
- O(2^n) 指数
- O(n!) 阶乘
